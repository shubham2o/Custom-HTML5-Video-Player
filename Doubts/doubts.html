Video played Property

• The played property retuns a TimeRanges object.

• The TimeRanges object represents ranges of the video that has already been played (seen) by the user.

• A played range is a time-range of played video. The user gets several played ranges if he/she skips in the video.

• Note : This property is ready-only.

HTML Audio/Video DOM pause() Method

• The pause() method halts (pauses) the currently playing audio or video.

• Use the play() method to start playing the current audio/video.
